Biyard Index 백서

본 문서는 Biyard Index 의 설계 원칙·축 정의·산출식·검증 절차를 정리한 기술 문서입니다. Biyard Index 는 토큰증권 (Security Token) 을 평가하기 위해 설계된 Web3 전용 지표이며, 기존 신용평가가 다루지 못하는 온체인 발행 무결성·스마트컨트랙트 보안·지갑 분포·거버넌스 등을 정량 축으로 환산합니다.

1. 개요

Biyard Index 는 분산원장 기반 토큰증권의 신뢰도를 정량적으로 평가하는 표준 지표입니다. 발행사 재무제표·증권신고서 같은 전통 정보는 기존 신용평가 (한국기업평가·NICE·KIS 등) 의 영역이며, Biyard Index 는 이들이 다루지 못하는 Web3 고유 신뢰 신호 를 측정합니다.

설계 원칙 — 주 입력은 온체인 데이터·컨트랙트 분석·거래소 시장 데이터. 공시 자료 (DART·EDGAR) 는 운영 주체·법적 구조 식별을 위한 보조 자료입니다.

1.1 기존 평가가 보지 못하는 영역

  • 발행 수량이 실제 신탁 자산과 1:1 로 일치하는가 (온체인 상태)
  • 토큰 컨트랙트에 발행사가 임의로 발행·소각할 권한이 있는가
  • 발행사·운영사 자체 지갑이 전체 토큰의 몇 % 를 보유하는가
  • 업그레이드 권한이 멀티시그·타임락으로 보호되는가
  • 거래소 거래량 중 Wash trading 신호가 얼마나 있는가

이 신호들은 분산원장 위에 기록된 사실이며, 공시·재무제표로는 확인할 수 없습니다.

2. 평가 대상

주 대상은 토큰증권 (2027.01 자본시장법·전자증권법 개정안 시행 후) 입니다. 시행 전 과도기 발행물에는 제한적 평가를 적용합니다.

4. 신호 (Signal) 정의

각 축은 다수의 신호로 구성됩니다. 모든 신호는 분산원장·컨트랙트 코드·거래소 API 에서 자동 추출 가능한 사실 정보 만 사용합니다.

4.1 컨트랙트 보안 (AXIS 01)

  • S1.1 외부 감사 (CertiK·Trail of Bits·Quantstamp 등) 수행 여부
  • S1.2 감사 리포트의 High/Critical 이슈 조치 비율
  • S1.3 소스코드 공개·검증 여부 (Etherscan Verified 등)
  • S1.4 업그레이드 가능 컨트랙트의 경우 프록시·관리자 권한 구조
  • S1.5 알려진 취약 패턴 (reentrancy·delegatecall·tx.origin) 검출 여부

4.2 온체인 발행 무결성 (AXIS 02)

  • S2.1 총 발행량 = 신탁/공시 명목 자산 수량 1:1 일치 여부
  • S2.2 발행자 임의 추가 발행 (mint) 권한 존재 여부
  • S2.3 소각 (burn) 이력의 공시 정합성
  • S2.4 이벤트 로그와 발행사 공시의 시점·수량 일치도

4.3 지갑 분포·집중도 (AXIS 03)

  • S3.1 상위 10 / 100 지갑이 보유하는 비중 (HHI 변형)
  • S3.2 발행사·운영사·관계자 식별 지갑의 자체 보유 비중
  • S3.3 활성 보유자 수 (90일 내 이동 이력 있는 지갑)
  • S3.4 잠금 (lock-up) 적용 지갑 비중·해제 일정 공개 여부

4.4 유통 신뢰성 (AXIS 04)

  • S4.1 거래소 (KRX·NXT 등) 일평균 거래량 / 시가총액 비율
  • S4.2 평균 호가 스프레드
  • S4.3 Wash trading 패턴 점수 (왕복 거래·자전 거래 탐지)
  • S4.4 가격 변동성 (vs 카테고리 평균)

4.5 거버넌스 (AXIS 05)

  • S5.1 업그레이드·중요 파라미터 변경의 멀티시그 적용 여부
  • S5.2 멀티시그 서명자 수·서명자 간 독립성
  • S5.3 타임락 (변경 예고 기간) 적용 여부·기간 길이
  • S5.4 운영 주체와 발행사·신탁사의 법적 분리 여부

4.6 정보 신뢰성 (AXIS 06)

  • S6.1 온체인 사건 (mint·burn·권한 변경) 과 공시의 시점 일치도
  • S6.2 운영사 채널 (홈페이지·소셜) 의 갱신 적시성
  • S6.3 이의 제기·문의에 대한 응답 시간·해결 비율
  • S6.4 외부 감사·평가 리포트의 공개 빈도

v1.0 백서에서 신호별 측정 방법·임계값·예외 처리까지 전면 공개합니다.

5. 산출식·가중치

5.1 축별 점수

AxisScorek = Σ (wi × Sk.i) × 100
단, Σ wi = 1

축 k 의 점수는 해당 축에 속한 신호 Sk.i 의 가중 평균을 100점 만점으로 환산한 값입니다.

5.2 종합 점수

TotalScore = Σ (Wk × AxisScorek)
단, Σ Wk = 1

축 가중치 Wk 는 토큰증권 유형별로 다르게 적용됩니다.

6. 등급 환산

종합 점수는 유형 내 백분위를 기준으로 다섯 단계 등급으로 환산됩니다.

등급유형 내 백분위설명
S상위 5%모든 축에서 평균을 크게 상회
A상위 5~25%대부분 축에서 평균 상회
B상위 25~60%유형 평균 수준
C상위 60~85%일부 축에서 평균 미달
D하위 15%다수 축에서 평균 크게 미달
등급은 같은 유형 내 상대 평가 입니다. 새 발행물이 추가되거나 시장 상황이 변하면 등급은 재산출됩니다.

8. 갱신·검증

  • 갱신 주기 — 온체인 신호는 일별 갱신, 거래·시장 신호는 시간별 갱신, 종합 점수·등급은 분기 1회 재산출 (주요 사건 발생 시 즉시).
  • 버전 관리 — 산출식·가중치 변경 시 백서 버전을 올리고 변경 이력을 공개.
  • 이의 제기 — 발행사가 산출 결과에 이의를 제기하면 입력 데이터 검증 후 재산출. 이의 제기 이력은 공개.
  • 독립성 — 평가 신청료·구독료는 받지만 등급을 변경하는 대가는 받지 않습니다. 모든 신청·산출 이력을 공개합니다.

9. 면책 조항

Biyard Index 는 자본시장법상 신용평가업·투자자문업·금융투자업이 아닙니다. 본 지표의 등급·점수는 온체인 데이터 및 공시 자료 기반 정보 제공 목적이며, 특정 토큰증권에 대한 투자 권유·매수 추천·목표가 제시가 아닙니다.

투자 결정은 사용자 본인의 책임이며, Biyard 는 본 지표를 근거로 한 투자 결과에 대해 어떠한 법적 책임도 지지 않습니다. 본 지표를 외부 채널에 인용·임베드할 경우 위 면책 조항을 함께 표기하셔야 합니다.
