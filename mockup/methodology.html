BIYARD INDEX

Biyard Index Methodology

Biyard Index는 STO 기초자산과 발행사 데이터를 5-Pillar로 정량화하는 지표입니다. 공식과 가중치는 백서 기준을 따르며, 데이터 출처와 미확보 항목은 산출 근거에 함께 표시됩니다. 검증 데이터가 추가되면 같은 공식 안에서 입력값이 보완되어 산출값이 달라질 수 있습니다.

왜 Biyard Index인가

기존 평가의 빈칸

전통 평가는 재무제표와 공시를 중심으로 봅니다. Biyard Index는 여기에 트레저리, 홀더 분포, 토큰 활동, 매출 오라클 같은 Web3 입력 데이터를 함께 반영합니다.

선택형 데이터 보강

공개·제출 자료만으로도 Index 산출은 가능하지만, Dataroom은 CT·MR의 신뢰도를 높이고 LaunchPad는 TB·CP·EG 입력값을 검증 가능한 형태로 보강합니다.

근거 중심 표시

Index 화면은 점수만 보여주지 않고 어떤 데이터가 확인되었고 어떤 항목이 미확보인지 함께 표시합니다.

최종 점수 공식

Score = 0.25 · TB + 0.25 · CP + 0.20 · EG + 0.15 · CT + 0.15 · MR
각 Pillar를 0~100으로 정규화 후 가중합

등급 매핑

등급점수 구간의미
T180~100입력 데이터 신뢰도와 운영 성숙도가 매우 높음.
T265~79대부분의 핵심 입력 데이터가 확인됨.
T350~64주요 입력 데이터는 확인되나 보완 항목 존재.
T435~49핵심 입력 데이터 일부 미확보.
T5~34공개 지표 산출에 필요한 데이터가 부족함.

5-Pillar 상세

TB · Treasury Backing 25%
스테이블코인 트레저리 충당 수준
발행 후
  • TB1: Treasury / Market cap
  • TB2: 월간 순적립 속도
  • TB3: 약속 준수율 (실제 / 약속)
  • TB4: 월별 발생률
  • TB5: 사전 공지 비율
CP · Community Power 25%
홀더 분포·홀딩 지속성
발행 후
  • CP1: 30일 잔고 >0 활성 홀더 수
  • CP2: Gini⁻¹ 분포 평등성
  • CP3: 평균 홀딩 기간 (개월)
  • CP4: 월간 신규 홀더 유입률
EG · Engagement 20%
토큰 활성 사용·거버넌스 참여
발행 후
  • EG1: 월간 활성 토큰 사용자 (MAU)
  • EG2: 30일 리텐션 (D30)
  • EG3: 사용자당 TX 빈도
  • EG4: 거버넌스·커뮤니티 참여
CT · Cash flow / Traction 15%
매출·성장률·트래픽 — 발행 전 실측 가능
발행 전 ✓
  • CT1: 오라클 매출 (PG·ERP·회계 SaaS)
  • CT2: 분기별 성장률 (QoQ)
  • CT3: web2 MAU·앱 다운로드
  • CT4: 토큰 거래량·유동성 (발행 후)
MR · Market Readiness 15%
공시·감사·규제 — 발행 전 실측 가능
발행 전 ✓
  • MR1: 공시 빈도
  • MR2: 감사사 수
  • MR3: 고객확인·AML 준비도
  • MR4: 락업 등 토큰 이코노믹스 공개
