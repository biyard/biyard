서비스 역할

Assets · LaunchPad
① Biyard Assets
  • 기초자산 탐색 (공개)
  • 기업 가입·인증 (B2B 테넌트)
  • 발행사 정보·기초자산 등록 (인증)
  • Data Room·클라우드 에이전트 (인증)
  • Biyard Index 산출의 단일 주체
  • 산출된 Index 공개 노출
② Biyard Launchpad
  • 토큰 발행·라이프사이클 관리
  • 트레저리·Oracle·온체인 운영
  • 홀더·커뮤니티·Web3 위젯
  • TB·CP·EG Pillar 데이터의 단일 공급원
  • Index 산출은 하지 않음 (데이터만 Assets로 전송)
서비스 구조
  • Assets — 기업 평가와 Biyard Index 산출
  • LaunchPad — 토큰 발행·운영·트레저리 관리
  • 토큰이 없는 기업도 Assets 평가 진행
  • 토큰 연결 시 TB·CP·EG 입력값 보강
